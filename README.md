# EventEmitter

Study 'Event' in Node.js.  
Thus, write various 'EventEmitter'.

Occur event using Custom Event and System Event.

## Custom Event
```
> node app.js
```
```
Okman event occurred.
Hello! I'm Okman.
```


## System Event
```
> npm i --save events
```

```
> node app2.js
```

```
Okman2 event occurred.
Hello! I'm Okman2.
```
